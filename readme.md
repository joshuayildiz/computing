# computing

Here you will find documents about my setup.  It is supposed to
only run on my machine so if anything does not, please do not @ me,
although feel free to ask questions through issues.

I primarily use [Acme] from [plan9port](https://github.com/9fans/plan9port)
and MacOS. My setup is as simple as installing those two and getting
a couple compilers off the internet. That's it.

I'm also a huge _typophile_, meaning I like fonts. Right now I kinda
am okay with [Berkeley Mono](https://usgraphics.com/products/berkeley-mono)
for when I need a fixed-width font and [Input Sans](https://input.djr.com)
for everything else (including code).

You will also notice a `Brewfile` for stuff I use. I don't actually
have any use for this file other than I like looking at how small
it is. I really don't need much when coding apparently...

The `src` directory is full of little scripts and stuff I use apart
from the `install` script, which really just installs the others.

I'd say do not look at `journal`.
